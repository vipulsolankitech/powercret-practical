Task Details

Task 1: Custom Post Type - Testimonials

A custom plugin named Testimonial was created to manage testimonials.

Implementation Details

  • Created a custom plugin with a proper plugin header.
  • Added an ABSPATH check to prevent direct access.
  • Registered the testimonial custom post type using register_post_type().
  • Created custom_fields.php for custom fields.
  • Added three meta fields using add_meta_box().
  • Saved fields using the save_post hook.
  • Added custom columns to the admin listing.
  • Made columns sortable.
  • Created a shortcode using add_shortcode().
  • Retrieved data using WP_Query.
  • Displayed testimonials on the /testimonial-front page.

Task 2: WooCommerce Gift Wrap Add-on

A custom plugin named Gift Wrap for WooCommerce was developed.

Implementation Details

  • Installed and configured WooCommerce.
  • Added a Gift Wrap checkbox on the cart page.
  • Added the gift wrap amount using add_fee().
  • Enqueued scripts using wp_enqueue_scripts.
  • Passed AJAX variables including nonce and admin-ajax URL.
  • Handled requests using wp_ajax_*.
  • Stored values in WooCommerce sessions.
  • Updated cart totals dynamically.
  • Reset the option using woocommerce_cart_emptied.
